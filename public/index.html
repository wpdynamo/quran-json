🔗 نقاط النهاية (Endpoints)

البيانات الوصفية:
/assets/quran/manifest_multi.json
فهرس الأجزاء (30 جزء):
/assets/quran/index_juz.json
فهرس الصفحات (604 صفحة):

السور (من 001 إلى 114):
/assets/quran/s{رقم السورة بثلاثة أرقام}.json
مثال: /assets/quran/s001.json (الفاتحة)

📋 مثال على البيانات

[ { "s": 1, "a": 1, "ar": "بِسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ", "en": "In the name of Allah, the Entirely Merciful...", "m": { "juz": 1, "hizb": 1, "rub": null, "page": 1, "ruku": 1 } } ]

🔑 شرح الحقول

  • s - رقم السورة
  • a - رقم الآية
  • ar - النص العربي (الرسم العثماني)
  • en - الترجمة الإنجليزية
  • m - معلومات إضافية (الجزء، الصفحة، الركوع)
