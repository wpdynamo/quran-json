🔗 نقاط النهاية (Endpoints)

البيانات الوصفية: /assets/quran/manifest_multi.json
الإحصائيات: /assets/quran/stats.json
معلومات السور (الأسماء، عدد الآيات، مكان النزول): /assets/quran/chapters.json
أسماء الله الحسنى (99 اسم): /assets/quran/allah_names.json
قائمة القراء: /assets/quran/reciters.json
فهرس الأجزاء (30 جزء): /assets/quran/index_juz.json
فهرس الصفحات (604 صفحة): /assets/quran/index_pages.json
السور (من 001 إلى 114): /assets/quran/s{رقم السورة}.json

مثال: /assets/quran/s001.json (الفاتحة)

📋 مثال على البيانات

[ { "s": 1, "a": 1, "ar": "بِسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ", "en": "In the name of Allah, the Entirely Merciful...", "m": { "juz": 1, "hizb": 1, "rub": null, "page": 1, "ruku": 1 } } ]

🔑 شرح الحقول

  • s - رقم السورة (من 1 إلى 114)
  • a - رقم الآية داخل السورة
  • ar - النص العربي (الرسم العثماني)
  • en - الترجمة الإنجليزية (Sahih International)
  • m - معلومات إضافية (الجزء، الحزب، الصفحة، الركوع)
